# KaalisBank

A digital banking demo I built to showcase front-end skills for banking/fintech internship applications. It simulates a UEMOA digital bank: a landing page with a working XOF/EUR/USD/GBP converter, a simulated login, and a dashboard with mock account data and a second converter.

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no backend. Typography uses IBM Plex Serif/Sans/Mono, and the design (navy + gold, sharp corners, two-tier nav) is inspired by real regional bank sites rather than generic templates.

This is a demo project only — not a real bank.

**Author:** Fatou, L1 GLSI student at ESP/UCAD, Dakar.
